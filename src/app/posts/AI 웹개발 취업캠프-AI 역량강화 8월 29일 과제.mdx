## 과제: 파이썬 코드의 문법을 정리
- 파이썬은 가독성을 위해 독특한 문법 구조를 가지고 있다.
#### 들여쓰기
- 파이썬은 블록 구조를 정의할 때 중괄호 대신 들여쓰기를 사용한다.
```py
if 조건:
    실행 코드
```
#### 콜론
- 조건문, 반복문, 함수 정의 등에서 콜론을 사용하여 코드 블록의 시작을 나타낸다.
```py
for 변수 in 리스트:
    실행 코드
```
#### 동적 선언
- 변수의 데이터 타입을 명시적으로 선언하지 않아도 된다.
- 파이썬은 실행 시간에 변수의 데이터 타입을 결정한다.
```py
x = 5  # 정수
y = "Hello"  # 문자열
```
#### 인덱스 슬라이싱
- 리스트나 문자열에서 요소를 접근할 때 인덱스를 사용하고, 슬라이싱을 통해 범위를 지정할 수 있다.
```py
my_list = [1, 2, 3, 4, 5]
item = my_list[2]  # 3
sub_list = my_list[1:4]  # [2, 3, 4]
```
#### elif
- 다중 조건문을 처리할 때 `elif` 키워드를 사용한다.
```py
if condition1:
    # 조건1이 참일 때 실행
elif condition2:
    # 조건2가 참일 때 실행
else:
    # 모든 조건이 거짓일 때 실행
```
