# AI 웹개발 취업캠프 역량강화-SW심화 FE 8월 9일 과제
<time>2023-08-14</time>

## 요약
- 본격적으로 리액트를 사용했다. 리액트 레이아웃의 특징, 컴포넌트 재사용성 등, 리액트의 장점을 학습했다. ```create-react-app```을 활용하여 로또 번호 추첨 프로그램 개발하며 실습했다.

## 실습 과정
- ```npx create-react-app my-project```를 활용해 프로젝트 생성
- 로또 번호의 *상태*가 변화할 때마다 컴포넌트를 재렌더링 해야하기 떄문에 ```useState``` 함수를 통해 자동으로 재렌더링 시킴
- 현재 날짜를 동적으로 받아오기 위해 ```new Data()```생성자를 사용
    - ```.getMonth()```를 통해 월을 가져올 시, 가져온 값에 1을 더해야 함에 주의
    - 서양 사람들은 월을 문자열로 표현하기 때문에, 배열 사용에 편의를 위해 값이 0부터 시작함
- 번호가 새겨진 공 모양의 엘리먼트를 반복해서 나열해야하기 때문에, ```map```함수를 사용
    - ```map```의 콜백함수의 두 번째 인자로 들어가는 인덱스를 공 모양의 엘리먼트에 key값으로 사용
- ```npm start```로 localhost 서버 실행
- 나만의 스타일로 CSS 수정
