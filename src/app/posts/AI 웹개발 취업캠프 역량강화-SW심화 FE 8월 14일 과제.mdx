# AI 웹개발 취업캠프 역량강화-SW심화 FE 8월 14일 과제
<time>2023-08-17</time>

## 요약
- 리액트 부트스트랩을 학습하고, 간단한 자기소개 페이지를 만들어봤다.

## 부트스트랩
- HTML, CSS, JavaScript 기반의 프론트엔드 프레임워크로 웹 디자인과 레이아웃을 빠르게 구축할 수 있게 해준다.
## 리액트 부트스트랩
- React 애플리케이션에 Bootstrap 디자인과 컴포넌트를 통합하여 UI를 개발하는 데 도움을 준다.

## 실습 과정
- 기존의 촌스러운 폰트에서 Google fonts가 제공하는 Noto Serif Korean 폰트로 바꿨다.
- 배경화면의 색을 밝은 회색으로 바꾸고 `radial-gradient`를 통해 점박 무늬를 추가했다.
- 기존의 Navbar 색을 투명하게 바꾸고 배경에 블러 효과를 추가했다. 그리고 그림자도 추가했다.
- 각 컨테이너간 간격을 `margin`과 `padding`을 통해 적절하게 넓혔다.
- 소개 세션의 이미지를 오른쪽에 붙이고, 테두리를 제거했다.
- 포트폴리오 세션의 배경색을 `dark`로 바꾸고 글자 색을 하얗게 바꿨다.
- 연락 세션의 제목 위치를 위쪽 세션과 더 멀어지게, 내부 콘텐츠와 더 가깝게 이동시켰다.
- 연락 이메일, 휴대전화, 소셜미디어 정보 엘리먼트의 테두리 색을 `dark`로 바꾸고, 배경색도 `dark`로 줬다. 추가로 상하좌우 `padding`을 더 추가했다. 글자 색을 흰색으로 바꿨다.
